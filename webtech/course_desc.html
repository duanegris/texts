Descriptif du cours
IT applications are increasingly web-based. Web technology has grown to include a variety of business, academic, organizational and social applications. This knowledge area covers the design, implementation and testing of web-based applications including related software, databases, interfaces and digital media. The course browses the main trends in web usages and describes technical issues and solutions. The course is illustrated throughout by concrete examples and a major part of the course is devoted to a personal project built by the student.

Objectif Pédagogique I -->Savoirs Acquis
Give an understanding of how current web technologies are used in IT applications today. Browse a panorama of web based IT applications to demonstrate their pervasiveness. Give an understanding of technical issues behind the scene.

Objectif Pédagogique II -->Compétences acquises
* Notions of algorithm design and programming * Server-side programming using PHP * Organization of information using relational databases * Implementation of a PHP-MySQL website * Installation of several collaborative web based solutions: Apache+PHP+MySQL, wiki, ...
